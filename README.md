#TCToolbox

The Cocktail Experience front tools

## Installing

- Install **[bower](http://bower.io/)**
- Install **grunt-cli** with **npm install -g grunt-cli**
- Execute **bower install**
- Execute **grunt**

## Updating packages

- Execute **bower update PACKAGE_NAME**
- Execute **grunt**

## Updating the theme

- Execute **bower update tct-foundation**
- Execute **grunt**

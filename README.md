#TCToolbox

The Cocktail Experience front tools

## Requirements

* [Node.js](http://nodejs.org)
* [Grunt](http://gruntjs.com/)
* [Bower](http://bower.io)

## Installing

- Execute **npm install**
- Execute **bower install**
- Execute **grunt**

## Updating packages

- Execute **bower update PACKAGE_NAME**
- Execute **grunt**

## Updating the theme

- Execute **bower update tct-foundation**
- Execute **grunt**
